Dataset Name: grammyWinners.csv

Data File Description: Data of grammy award winners on individual songs and records, so awards which exclude full albums. The data is collected over most all genres. A built function determines how many years of data to collect. There are 61 years of total data as the Grammy Awards have been held for 61 years to this point (7/6/19). 

Columns:
	-Award: grammy that was won
	-GrammyYear: year for which the award was won
	-Genre: genre the award was given under
	-SongTitle: title of the song
	-Artist: name of the artist


File Format: CSV or comma separated value file

Related Data Files: None

Data Owner: Recording Academy Grammy Awards

Contact Person/Primary Data Investigator: Daniel DeFoe; California Polytechnic State University San Luis Obispo, California; ddefoe@calpoly.edu

Data-Collection Date: after 7/6/2019
